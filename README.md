# 3D Raya
Juego tres en raya y tres en raya en 3D.

## 🛠️ Pre-requisitos
Para poder usar esta aplicación es necesario tener instalado **Java**.
* En Windows podrás ejecutarlo de manera sencilla con solo hacer doble click en el _.jar_.
* En Linux tendrás que emplear los comandos _java -jar ./tu_ruta/tresdraya.jar_ No olvides darle permisos.

## 🖇️ ¿Dónde lo puedo descargar?
Desde la página de [Games Jolt](https://gamejolt.com/games/3draya/342722).
